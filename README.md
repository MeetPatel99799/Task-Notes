# Express Task-Notes App

A simple notes application built with Node.js, Express, and EJS. This app allows you to create, view, and rename notes, with each note stored as a text file on the server.

## Features

- Create new notes
- View note contents
- Rename notes
- Clean and modern UI with Tailwind CSS
